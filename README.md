# 技術スタック

- ドメイン取得:Dot tk(Freenom)
  → 無料
- ネームサーバー:Cloudflare
  → 安全面と操作性のため
- メインサーバー:Oracle Cloud Always free osaka Region
  → 永年無料枠を使用、東京は混雑してるので大阪のサーバーに
- タスク管理:Asana+Instagantt
  → 無料でガントチャートが気持ちよく使える構成
- デザイン制作:Figma
- 議事録管理:Notion
- wiki:GitHub Wiki
- ローカル開発環境:Docker
- ブレインストーミングツール:hidane
- 連絡:Slack
- 会議:Discord
